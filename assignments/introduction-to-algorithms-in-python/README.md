# 📘 Assignment: Introduction to Algorithms in Python

## 🎯 Objective

Implement and analyze basic algorithms like sorting (bubble sort) and searching (binary search) to understand algorithmic thinking and efficiency.

## 📝 Tasks

### 🛠️ Implement Bubble Sort

#### Description

Write a function that sorts a list of numbers using the bubble sort algorithm.

#### Requirements

Completed program should:

- Define a function `bubble_sort(arr)` that takes a list and returns the sorted list
- Use bubble sort logic: repeatedly swap adjacent elements if they are in wrong order
- Include comments explaining each step

### 🛠️ Implement Binary Search

#### Description

Write a function that searches for a target value in a sorted list using binary search.

#### Requirements

Completed program should:

- Define a function `binary_search(arr, target)` that returns the index if found, -1 otherwise
- Assume the input list is sorted
- Use binary search logic: divide and conquer
- Include comments explaining each step
